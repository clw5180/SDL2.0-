### 1、说明
**利用SDL事件处理与绘图功能实现人物行走的小例子。我自己用Mov Gear软件做的Sprite Sheet，里面包含了八个方向的静止、行走和跑动图；目前很容易就可以加入跑动效果，只需要在事件处理类CInputHandler中加一个shift键，然后把每次坐标步进值调大即可。 **
