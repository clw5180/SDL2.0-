### 1、说明
一个支持SDL输出中文，外加碰撞检测的小例子。

### 2、注意事项
- 链接器-输入-附加依赖项要包含SDL2相关的lib文件：
  - SDL2.lib
  - SDL2main.lib
  - SDL2_image.lib
  - SDL2_mixer.lib
  - SDL2_ttf.lib
- 最后要把SDL2相关的dll文件放在Windows系统目录或exe同一目录下！
